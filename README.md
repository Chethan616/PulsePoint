A Flutter-based mobile application connecting blood donors with recipients in real-time. Built with Firebase for seamless integration and real-time updates.

## Features

### Core Functionality
- 🩸 User authentication with Firebase
- 📍 Geolocation-based donor search
- 🔍 Advanced filtering (blood type, distance, age)
- 💬 Real-time chat functionality
- 📲 Blood request management system
- 📨 Contact form with Firestore integration

### Key Screens
- **Registration Flow**: Secure user onboarding
- **Donor Discovery**: Interactive map/list view
- **Blood Request System**: Create/manage requests
- **Chat Interface**: Real-time communication
- **Profile Management**: User health data tracking

## Technology Stack
- **Frontend**: Flutter (Dart)
- **Backend**: Firebase (Auth, Firestore, Storage)
- **Maps & Location**: Geolocator API
- **State Management**: Provider
- **CI/CD**: GitHub Actions

## Installation

### Prerequisites
- Flutter 3.13.8+
- Dart 3.1.0+
- Firebase project setup

### Setup
1. Clone repository:
2. flutter pub get
3. flutter run
